Política de Privacidad

Esta política explica cómo nuestra aplicación recopila, usa y protege los datos del usuario.

1. Datos Recopilados

- Notificaciones Push: Usamos Expo Notifications para enviar alertas sobre actualizaciones de Fortnite. No recopilamos datos personales.
- Enlaces externos: La aplicación permite abrir artículos del sitio oficial de Fortnite. No rastreamos ni almacenamos información del usuario.

2. Finalidad del uso de datos

Los datos se usan únicamente para:
- Enviar notificaciones sobre actualizaciones del juego.
- Acceder a contenido oficial externo.

3. Almacenamiento de Datos

No almacenamos datos personales. Las configuraciones locales se guardan en el dispositivo del usuario.

4. Compartición con terceros

No compartimos datos con terceros. La aplicación solo muestra contenido público de fortnite.com sin seguimiento.

5. Derechos del Usuario

Según el RGPD y la CCPA, los usuarios pueden:
- Solicitar información o eliminación de datos (no recopilados).
- Desactivar las notificaciones desde la configuración del dispositivo.

6. Contacto

Para dudas sobre privacidad, contáctanos en: fortniteupdatetracker@gmail.com
